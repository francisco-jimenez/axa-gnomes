## Deployment + Service start  

git pull  
npm build  
serve -s build -l 3000  

## AWS Setup (Amazon Linux 2)  

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash  
nvm install node  
npm install -g serve
